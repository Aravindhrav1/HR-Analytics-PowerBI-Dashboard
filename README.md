# HR-Analytics-PowerBI-Dashboard
Interactive HR Analytics Dashboard built using Power BI, Power Query and DAX to analyse employee attrition, performance and engagement trends.

## Project Overview:
This project was developed using Power BI to analyse employee attrition, performance, engagement and workforce wellbeing.

The objective was to transform raw HR data into interactive dashboards that help stakeholders understand workforce trends, identify potential risks and support evidence-based decision-making.

## Business Objectives:
- Analyse employee attrition trends
- Understand performance and rating patterns
- Monitor employee engagement and satisfaction
- Identify burnout and retention risks
- Support workforce planning through data-driven insights

## Tools & Technologies:
- Power BI
- Power Query
- DAX
- Data Modelling
- Excel

## Key Insights:
- Sales department recorded the highest attrition rate.
- Younger employees demonstrated higher turnover risk.
- Employees consistently rated themselves higher than managers.
- Technology teams showed stronger engagement levels.
- Burnout risk was higher among employees working overtime.
